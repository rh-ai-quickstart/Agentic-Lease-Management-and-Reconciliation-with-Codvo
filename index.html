NeIO LeasingOps Helm Charts

Helm charts for deploying NeIO LeasingOps on OpenShift and Kubernetes.

Quick Start

# Add the Helm repository
helm repo add neio-leasingops https://codvo-ai.github.io/neio-leasingops-quickstart
helm repo update

# Search available charts
helm search repo neio-leasingops

# Install a chart
helm install leasingops neio-leasingops/neio-leasingops

Available Charts

neio-leasingops

Main LeasingOps application chart with AI-powered aircraft leasing operations platform.

neio-openshift-ai

OpenShift AI (RHOAI) integration for model serving and inference.

neio-pgvector

PostgreSQL with pgvector extension for vector storage.
